Ssh : Secure Shell 다른 컴퓨터에 로그인 또는 원격 명령을 해주는 프로그램과 프로토콜 TCP번호 22번
Telnet : 인터넷이나 로컬 영역 네트워크 연결에 쓰이는 프로토콜
Http : world wide web 상에서 정보를 주고받을 수 있는 프로토콜
Https : http over secure socket layer 보안이 강화된 http 프로토콜
Ftp : TCP/IP 프로토콜을 가지고 파일 전송을 하기 위한 프로토콜
Sftp : 보안이 강화된 ftp프로토콜
SMTP : 메일 전송 프로토콜 TCP번호 25번
POP3 : 이메일에 접속할 때 사용하는 프로토콜
IMAP : 인터넷 메시지 접속 프로토콜
NNTP : 네트워크 뉴스 전송 프로토콜 기사 구독과 게시에 쓰임
DNS : 도메인 네임 시스템 호스트 도메인 이름을 네트워크 주소로 바꾸도록 함
